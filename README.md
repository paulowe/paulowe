The code that powers [paulowe.com](https://paulowe.com).

## To set up in Development
Clone the repository:
`git clone git@github.com:paulowe/paulowe.git`
`cd` into the directory:
`cd paulowe`

Install dependencies:
`yarn`

Start the client:
`yarn dev`

Open the site:
`localhost:3000`
